Usage
  • The data is updated every few seconds, re-open the popup window to see fresh data.
  • The time watched doesn't update if the tab is not visible (it's essentially paused until visible again.)
  • The extension will add new bars in the chart and adjust scales and layout automatically as you watch more streams.
Troubleshooting tips:
  • If you have a Twitch tab open when installing the extension it might cause issues, reload the tab.
  • If that doesn't solve it, restart Chrome.
  • Finally, try reloading the extension.
Make sure you re-open this popup window after clicking the Clear data button
